Welcome to my personal space.

This space was created to introduce myself and as a demonstration of the technologies that I can work with to create presence on the internet for individuals or businesses.

I have been a developer for almost 10 years now. And during that time I haved worked with several different technologies to create web sites, applications, web services and mobile applications. My primary focus has been Java development (for the enterprise, mostly backend) but I have had the oportunity to interact with several other web technologies and some andriod development.

Recently I have felt the need to start experimenting with some front-end technologies hence the creation of this website. I wanted to train myself in tools and techonologies like Node, npm, Jekyll, HTML 5, CSS 3, JavaScript, Bootstrap, Sass, etc. and I thought this would be a perfect way to do it.

If you feel like experimenting with projects like this, feel free to clone the repository and start changing whatever you need to fit your needs. As a good open source developer, I will give you anything that you want from this website for free. To know more about how this project was built checkout the README.md file of the project.
